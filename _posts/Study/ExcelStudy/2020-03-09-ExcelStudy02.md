# 단축키

- F5 : 이동하기 위한 단축키!!
- CTRL : 빠르게 이동
- Shift : 선택
- Ctrl + D : 위의 값으로 아래 값 모두 적용(Down)
- Ctrl + R : 오른쪽에 내 값들을 적용!(right)
- 시트 이동 시! : 왼쪽아래에서 오른쪽 마우스 버튼 누르면 활성화창 나옴!
- Ctrl + enter : 동시입력
- 근데 빈칸이 엄청 많아 그럼 어떻게 해?! : F5 눌렀을 때, 옵션 키 나와! 그걸로 이용하면 쉬워
- F4 : 절대 참조! 단축키
